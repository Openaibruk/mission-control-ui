# 📡 Nova Tech Radar

## 🧠 Core Agent Tech & Memory
* **Infinite Long-Term Memory:** Tom Dörr highlighted [ContextKeep](https://github.com/mordang7/ContextKeep), a new tool for giving AI agents infinite long-term memory.
* **Fixing the "Blank Slate" Problem:** Atenov published a guide on how to stop your AI agent from waking up with a blank memory every session. He breaks down the one config change that makes context and business knowledge stick permanently.
* **Agents Running the Back-Office:** Pontus Abrahamsson introduced the Midday CLI, a backbone for AI agents containing 80+ tools for invoicing, reconciliation, exports, time tracking, and reports. 
* **The Motherload of OpenClaw Templates:** The `awesome-openclaw-agents` repository was shared, featuring 162 production-ready AI agent templates (complete with copy-paste `SOUL.md` configs).

## 🏢 Real-World AI Workflows
* **Living with an Agent:** Witcheer shared their full setup after 2 months of living with an AI agent. It includes 7 AM Telegram briefings summarizing overnight market moves, arXiv papers, Hacker News threads, and r/LocalLLaMA posts.
* **Replacing $400K in Roles:** OSF dropped a step-by-step guide on how they "gave Claude a brain" to run operations at their company, RektDrinks, replacing massive overhead costs. 

## 🛠️ New Tools & Skills
* **Gemini 3.1 Flash Live API:** Google AI Studio launched the new Gemini Live API, enabling developers to build real-time voice and vision agents with incredibly low latency.
* **New Agent Skills:** Several new open-source agent skills hit the timeline, including an [AI Marketing Suite](https://github.com/zubair-trabzada/ai-marketing-claude) for Claude Code, a repo of [SEO and GEO skills](https://github.com/aaron-he-zhu/seo-geo-claude-skills), and a database of [10,000+ curated AI image prompts](https://github.com/YouMind-OpenLab/ai-image-prompts-skill).
* **MoneyPrinterV2:** Sami Hindi dropped MPV2 (MoneyPrinterV2), an open-source tool for automating the creation of YouTube Shorts and TikToks.
