## ElevenLabs Setup

1. Go to elevenlabs.io → Sign up (free tier = 10k chars/month)
2. Voice Design → Create custom voice:
   - For Sara: "Young American Girl" — modify to sound closer to East African accent
   - For Kaleb: "Young Child" — higher pitch, playful
   - For Ato Gobeze: Create with voice cloning — record 30 seconds of Amharic with a native speaker
3. Stability: 40-50% (more expressive), Similarity: 75%
4. Type each line separately — export as .wav
5. Name: `sara_line_01.wav`, `kaleb_line_01.wav`, etc.

## Narakeet Setup (fallback)
1. Go to narakeet.com
2. Select language: Amharic
3. Paste Amharic text lines directly
4. Download audio files
