# The Fidel Song — Complete Script
## Pilot 01 | Ethiopian Kids Animation Channel

**Length:** ~3 minutes  
**Target Age:** 2–8 years  
**Format:** Musical / Phonics / Bilingual (Amharic + English)  
**Characters:** Sara (7yo, confident big sister), Kaleb (5yo, curious little brother), Ato Gobeze (funny Gelada baboon — appears as surprise cameo)

---

## Song Lyrics (Main Track)

### 🎵 Opening Hook (0:00–0:15)
**Tune:** Upbeat Ethio-jazz/Afrobeat groove

```
🇪🇹 Amharic:
ሀ ለ ሐ መ ሠ — ይዘፍኑ ቡዞች!
ለ ለ ወድድር ጋር — አብረን እንማር!

🇬🇧 English echo:
Ha La Ha Ma Sa — come and sing with me!
Let's learn our letters — together, happily!
```

### 🎵 Verse 1 — H Sound Family (0:15–0:50)
```
ሀ — ሀውልት ሀውልት! (Obelisk, obelisk!)
   Stand so tall like an obelisk, ሀ ሀ ሀ!

ለ — ልጅ ልጅ! (Child, child!)
   Every child can learn and sing, ለ ለ ለ!

ሐ — ሐይል ሐይል! (Power, power!)
   Feel the power in your voice, ሐ ሐ ሐ!

መ — ማር ማር! (Honey, honey!)
   Sweet like honey, taste it now, መ መ መ!

ሠ — ሠላም ሠላም! (Peace, peace!)
   Peace and love for everyone, ሠ ሠ ሠ!
```

### 🎵 Bridge — Ato Gobeze Interlude (0:50–1:20)
```
🐒 Ato Gobeze (funny voice):
"Wait wait wait! You think you know the H sound?
Try THIS in Amharic — ሀ ለ ሐ መ ሠ!
Now dance with me — አንድ ሁለት ሦስት! (1, 2, 3!)"
```

### 🎵 Verse 2 — Dance & Repeat (1:20–2:00)
*All 5 letters repeated with choreographed animation*
```
Sara: ሀ!
Kaleb: ሀውልት!
Both: ሀ ሀ ሀ!

Sara: ለ!
Kaleb: ልጅ!
Both: ለ ለ ለ!

*(repeats for ሐ, መ, ሠ with corresponding words)*
```

### 🎵 Chorus — The Fidel Hook (2:00–2:30)
```
🇪🇹 ሀ ለ ሐ መ ሠ — these are the sounds!
🇪🇹 አብረን እንማር — Amharic is fun!
🇬🇧 Sing it loud, sing it clear,
🇬🇧 Fidel letters, we're learning here!
```

### 🎵 Outro (2:30–3:00)
```
Sara: You did it Kaleb!
Kaleb: I learned the H family!
Ato Gobeze: Next time — more letters! እስኪ እንጂ!
Both: ቻው! (Bye!)
```
