## 🎯 Core Voice Principles

### 1. Clarity First
- Every sentence earns its place
- Cut the fluff — get to the point
- Use plain English; avoid jargon unless it's industry-standard

### 2. Confident (Not Arrogant)
- State things directly: "We help teams..."
- Avoid: "We think we might possibly help..."
- Own our expertise without bragging

### 3. Helpful to the Core
- Lead with value, not features
- Ask: "Does this help the reader?"
- Solve problems, don't just describe them

### 4. Human, Not Corporate
- Write like you talk (professionally)
- Contractions are fine: "we're," "you'll," "don't"
- Avoid: "Please kindly revert" — just say "let me know"

---

## ✍️ Writing Rules

### Do ✓
- Use active voice
- Keep sentences under 25 words when possible
- Start with the benefit, not the feature
- Use numbers: "3 ways" not "several ways"
- End with action: "Try this →" not "This could be considered"

### Don't ✗
- Use filler words: "basically," "actually," "literally"
- Write passive voice: "Improvements have been made" → "We improved"
- Use corporate-speak: "leverage," "synergy," "drill down"
- Bury the CTA
- Say "in order to" — just say "to"

## 🚀 Hook Formulas That Work

### For Headlines
- **The "Number + Benefit" Hook:** "5 Ways to [Desired Outcome]"
- **The "Reverse" Hook:** "Stop Doing [Common Mistake]"
- **The "Question" Hook:** "Struggling with [Problem]?"
- **The "Bold Claim" Hook:** "[Controversial Take]"

### For Openers
- "Here's the thing most people miss about..."
- "You don't need [expensive solution] to..."
- "The [industry] is wrong about..."

## 🔤 Word Bank

### Words We Love
- Actually, really, simply, just
- Build, create, solve, fix
- Try, test, experiment
- Results, impact, outcome

### Words to Avoid
- Leverage, incentivize, paradigm
- Synergy, ecosystem, holistic
- Revolutionary, game-changing (unless true)
- Best-in-class, world-class (empty flattery)

## 📝 Copy Templates

### Feature Description
```
[Problem they have] + [Our solution] + [Result they get]

"Managing multiple tools slows you down. Our unified dashboard brings everything to one place — so you ship faster."
```

### CTA Button
```
[Action] + [Benefit]

"Start your free trial" (not "Submit")
"See how it works" (not "Click here")
"Get early access" (not "Register now")
```

### Email Subject Line
```
[Curiosity] + [Specific]

"Here's what most marketers miss about..."
"The 3-email experiment that doubled our conversions"
```

## 🔍 Voice Check

Before publishing, ask:
- [ ] Is this clear in 3 seconds?
- [ ] Does it sound like a human wrote it?
- [ ] Is there a clear next step/CTA?
- [ ] Would I say this in a conversation?
- [ ] Is every sentence necessary?
