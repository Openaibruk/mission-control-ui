# Quick-Start Checklist: Ethiopian Kids Animation Channel

## Week 1–2: Foundation
- [ ] Finalize channel name (“Tibeb Kids” option)
- [ ] Design 3 core characters (concept art → Blender models)
- [ ] Set up YouTube channel (branding: logo, banner, description)
- [ ] Write 5 pilot scripts (phonics, numbers, hygiene)
- [ ] Set up Amharic TTS (Google Cloud or Play.ht) and test audio output

## Week 3–4: First Episode
- [ ] Rig main character in Blender (full face/body)
- [ ] Animate 60-second Short using AI lip sync + motion cycles
- [ ] Record voiceover, clean audio
- [ ] Add music, captions (EN/AM), export
- [ ] Upload privately; get feedback from 2–3 parents

## Week 5–6: Iterate & Scale
- [ ] Revise based on feedback
- [ ] Build background asset library (school, home)
- [ ] Batch produce 2 more Shorts
- [ ] Create first 2–3 min long-form episode (compilation of Shorts + new content)
- [ ] Publish schedule: 3 Shorts/week, 1 long-form/month

## Week 7–8: Growth & Monetization
- [ ] Optimize titles/thumbnails; add playlists
- [ ] Set up AdSense
- [ ] Join YouTube Kids (if eligible)
- [ ] Share to 3 Ethiopian parent communities
- [ ] Plan merch (printable PDFs) or licensing pitch
