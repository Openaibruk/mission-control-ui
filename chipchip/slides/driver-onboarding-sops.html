SOP-001 New Driver Registration

📥 Application Receipt

  • Driver submits via app or in-person
  • System generates Driver ID (CC-XXXX)
  • Confirmation SMS/email sent

🔍 Document Verification (24h)

  • Driver's license authenticity
  • Vehicle registration validity
  • Insurance coverage check
  • Background check via approved provider

⚖️ Eligibility Decision

  • Approved: → Account setup
  • Conditional: Request missing docs
  • Rejected: Notification + appeal process

⚙️ Account Provisioning

  • Create driver profile
  • Assign delivery zone
  • Set tier to Bronze
  • Enable training module access

🚨 Escalation Path

Doc issues → Ops Lead (4h) • Background flags → Compliance (immediate) • System errors → Tech Support (1h)

SOP-002 Training & Assessment

📚 Training Access

  • Send app download + training login
  • Schedule orientation (within 7 days)
  • Assign training materials

📊 Completion Tracking

  • Monitor via admin dashboard
  • Reminder if not started in 48h
  • Auto-unlock quiz after all modules

✅ Assessment

  • Quiz: ≥80% to pass
  • Failed: 24h cooldown → retake
  • Max 3 attempts → in-person review

🚗 Practical Assessment

  • Supervised delivery (recommended)
  • Eval: navigation, customer interaction
  • Document feedback in profile

📋 Training Modules

App navigation • Order management • GPS navigation • Customer communication • Delivery confirmation • Issue reporting • Package handling • COD procedures • Safety protocols

SOP-003 First Week Driver Support

📅 Day 3 Check-in

  • Automated: "How's it going?"
  • Review first delivery metrics
  • If no deliveries → phone call

📅 Day 5 Review

  • Pull: deliveries, avg rating, response
  • Rating < 3.5 → coaching call
  • 0 deliveries → investigate barriers

📅 Day 7 Week 1 Review

  • Comprehensive performance summary
  • Identify strengths & improvements
  • Set goals for week 2

SOP-004 Driver Activation

✅ Eligibility Check

  • All onboarding phases completed
  • Training quiz passed
  • Week 1 minimums met
  • No outstanding compliance issues

⚙️ System Updates

  • Remove "new driver" flag
  • Expand delivery zone access
  • Enable full features (peak hours, long-dist)
  • Add to active driver pool

👥 Community Onboarding

  • Add to driver WhatsApp/Telegram group
  • Invite to next community meetup
  • Assign mentor (Gold/Platinum tier)

🔔 Notifications

  • Send activation confirmation to driver
  • Notify team lead of new active driver
  • Log activation date for tracking

SOP-005 Incident Handling During Onboarding

⚠️ Incident Types

  • Document fraud detected
  • Safety violation during training
  • Vehicle inspection failure
  • Customer complaint (supervised delivery)
  • Driver injury during training

🚨 Immediate Response

  • Suspend onboarding process
  • Document incident details
  • Notify Ops Manager within 1h
  • Ensure driver safety (medical if needed)

SOP-006 Driver Exit Process

🟢 Voluntary Exit

  • Receive notice (app/phone/in-person)
  • Exit interview
  • Final payment (within 7 days)
  • Deactivate + reclaim materials

🟡 Involuntary (Performance)

  • Document issues with evidence
  • 2 written warnings (7-day intervals)
  • Final review with Ops Manager
  • Deactivate + final payment

🔴 Involuntary (Violation)

  • Immediate suspension
  • Investigation within 48h
  • Reinstatement or termination
  • Final payment (minus penalties)

Post-Exit Checklist

☐ Remove from all driver groups

☐ Reclaim physical materials (ID, stickers)

☐ Archive records (retain 2 years)

☐ Update driver database

UX Review — Friction Points

🔴 Critical Issues

  • Document Upload Bottleneck — 7 docs required upfront, no save-and-resume. Fix: partial registration within 48h
  • No Mobile Scanning — Pre-scanned docs required. Fix: in-app camera + OCR
  • Quiz 80% Threshold — No retake policy. Fix: unlimited retakes with cooldown

🟡 Moderate Issues

  • Rigid 5-day check-in schedule → flexible windows
  • No self-service progress tracking → progress bar
  • Ambiguous background check timeline → 24-48h expectation
  • Welcome kit timing unspecified → ship Day 1
  • No peer mentor → assign buddy driver
