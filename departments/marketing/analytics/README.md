**File:** `campaign-report-template.md`

Fill-in-the-blanks template for analyzing individual campaigns. Includes:

- Creative performance analysis
- A/B test results section
- Cost analysis
- Learnings & recommendations

### 3. KPI Dashboard Reference
**File:** `kpi-dashboard-reference.md`

Quick-reference guide for all marketing KPIs. Includes:
- Golden metrics (LTV:CAC, CAC Payback, ROAS)
- Acquisition, Revenue, Engagement, Retention KPIs
- Channel-specific targets
- Funnel benchmarks
- Weekly/Monthly checklists
- Dashboard links
- Quick formulas
